# Hacs

**警告：本文書はAIによって生成されたものであり、内容の正確性は保証されません。正確な内容については中国語版を参照してください。**

Hacsは、マウスを使用してショートカットキー操作を実現するツールです。

Hacsの名前は「Hexagon-alt-ctrl-shift」の略で、六角形を主体としたUIと3つの修飾キーに対応しています。

## 使用方法

Hacsを使用してショートカットキーを送信する手順は以下の通りです（QQスクリーンショットショートカットCtrl+Alt+Aを例にします）：

1. Hacsを起動します；
2. スクリーンショットを撮りたい画面で右クリックして下にドラッグします。この時、画面上に底辺がx軸に平行で頂点が上を向いた青緑色の半透明の二等辺三角形が表示されます。右クリックを離します；
3. 右クリックを離すと、画面上にいくつかの青緑色の半透明の六角形が表示されます。その中から「A」の文字がくり抜かれた六角形を見つけ、左クリックするとショートカットキー操作が完了します。

## 設定

### キーコンテンツ

インターフェースの「Add」ボタンまたは「Modify」ボタンを使用してショートカットキーの内容を更新する際は、以下のルールに従ってください。

1. 大文字のアルファベットは対応する文字キーを表します。例えば、「A」はキーボードのAキーを表します。
2. 数字と記号は、メインキーボードエリア上の対応するキーを直接表します。シフトモードではない状態で表示される文字に基づきます。標準配列を例にすると、「`1234567890-="は、バックティック、数字、ハイフン、イコールを順に押すことを表します。
3. その他のキーについては、「{キー名}」の形式を使用してください。一部の一般的なキーには、小文字の単一文字形式が提供されています。例えば、「{Enter}」と「e」はどちらもEnterキーを表します。詳細は以下の表を参照してください。

### 選択パネル

選択パネルは、右クリックを離したときに表示されるショートカットキーを選択するためのインターフェースです。

選択パネルのサイズは1000x1000で、左下を原点としてx軸とy軸が正の方向に伸びています。ショートカットキーのボタンのOffsetXまたはOffsetYを編集するときは、選択パネルの中心に対するボタンの中心のオフセットを指します。ボタンは高さ100、全長200、上下の辺の長さが100の六角形です。(0,0)の位置には、対角線の長さが50の菱形のキャンセルボタンが配置されています。
